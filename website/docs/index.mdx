title: Kalshi CLI
---

`kalshi-cli` is a terminal-first interface for Kalshi trading and market discovery.

- Uses the official TypeScript SDK: `kalshi-typescript`
- Prioritizes practical command workflows for traders
- Supports table output by default with `-o wide` and `--json`
