`kalshi-cli` is a terminal-first interface for Kalshi trading and market discovery.

- Uses the official TypeScript SDK: `kalshi-typescript`
- Prioritizes practical command workflows for traders
- Supports table output by default with `-o wide` and `--json`

## Recommended Install

```bash
npm install -g @anmho/kalshi-cli
kalshi --help
```

The package name is scoped, but the executable is still `kalshi`.

## Auth At A Glance

Authenticated commands need:

- `KALSHI_API_KEY_ID`
- `KALSHI_PRIVATE_KEY_PATH` or `KALSHI_PRIVATE_KEY_PEM`

Start with Kalshi's official API key guide:

- [Kalshi API keys guide](https://docs.kalshi.com/getting_started/api_keys)
